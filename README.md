# Bigfoot

Bigfoot is the missing (big) schema manager for Google BigQuery.

# Installation

```zsh
pip install bigfoot
```

# Usage

```zsh
bf pull {bq|sheets}
bf push [--dry-run] {bq|sheets}
```
